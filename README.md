## Documentación de maquinas virtuales 📋

En este repositorio se van a encontrar reportes de maquinas resueltas, la plantilla usada para la presentación de la infromación y documentación, maquinas resueltas y evidencias de lo encontrado. 
La plantilla se compone de 6 fases: 
- Objetivo
- Enumeración
- Explotación
- Post-explotación
- Lecciones
- Evidencia

## Explicacion de las faces
-- -

1️⃣**Objetivo**: Explicación de lo que se va a analizar o comprometer


2️⃣**Enumeración**: Recopilacion de lo que hay antes de atacar, puertos, tecnologias, servicios, etc...


3️⃣**Explotación**: Como se aprovecho la vulnerabilidad.


4️⃣**Post-explotación**: Que se hizo antes de entrar, para demostrar el impacto.


5️⃣**Lecciones**: Que se aprendio.


6️⃣**Evidencia**: Pruebas visuales de lo hallado.
